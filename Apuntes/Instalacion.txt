Instalacion Vue js 3
npm init vue@latest
cd vue-project
npm install
npm install -g @vue/cli
vue add vuetify (proyectos nuevos) Opcion: "vuetify 3 vite"

npm i vue-axios
npm install @http/auth

npm run dev

Añadir complemento Vue.js devtools a complementos del navegador
Añadir extensiones visual Studio:
    Vue Language Features (Volar)
    Vue 3 Snippets

Con "npm install" podemos añadir todas las dependencias en caso de no tenerlas.


Para errores globales en Vuetify:
npm install @vue/runtime-core @vue/reactivity @vue/shared
